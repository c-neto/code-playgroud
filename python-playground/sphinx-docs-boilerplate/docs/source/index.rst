Bem vindo a documentação do módulo *my-module*
=====================================================

Este módulo faz parte do conjuto de serviços que compõem o produto **my-module-enterprise**.

A documentação deste módulo está separada em duas seções. A seção denominada *Deploy*, é focada na configuração e execução do módulo em ambiente de produção. A outra seção, denominada *Desenvolvimento*, é voltada em informações e descrições dos elementos que diz a respeito a código fonte do projeto, concentrando a documentação das classes e submódulos juntamente com a descrição da arquitetura e as rotinas programadas.
